# clAIre 🤖
### AI Assistive Robot for Elderly Care

> Submitted to the Samsung Solve for Tomorrow Competition

---

## Overview

clAIre is an AI-powered assistive robot designed to improve the quality of life for elderly individuals by providing daily assistance, emergency support, health reminders, and companionship.

As the elderly population continues to grow, many seniors face challenges such as:
- Forgetting medications
- Limited mobility
- Isolation and loneliness
- Delayed emergency response
- Difficulty using modern technology

clAIre solves these problems through an accessible robotic assistant powered by artificial intelligence.

---

# Key Features

## 🗣️ Voice Assistant
- Natural voice conversations
- Answers questions and provides reminders
- Elderly-friendly interaction system

## 💊 Medication Reminders
- Tracks medication schedules
- Sends voice reminders
- Alerts caregivers if medication is missed

## 🚨 Emergency Detection
- Detects falls using AI and sensors
- Automatically alerts caregivers
- Emergency voice activation

## 🧭 Smart Navigation
- Autonomous indoor movement
- Obstacle avoidance
- Home assistance

## ❤️ Companionship
- Conversational AI companion
- Plays music and interacts socially
- Helps reduce loneliness
